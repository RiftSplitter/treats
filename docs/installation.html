Installation

In this chapter we'll learn how to setup our first Treats app, Treats is an easily extensible, feature-rich NodeJS+React Framework. With Treats, you don't need to configure webpack, babel, or any other build tools by yourself.

Quick Setup Guide

NOTE: Before proceed to the first step, please make sure you have installed [npm] and [yarn] on your PC.

  1. To start creating Treats project, install create-treats-app.
~$ npm install -g create-treats-app
  1. Run create-treats-app on your terminal. Enter app credential you need when prompted (optional).
~$ create-treats-app
  1. Wait the installation done and test your app. To test your app, go to your app directory and start the app.
~$ cd /YOUR/APP/DIRECTORY
~$ yarn start
  1. Open your browser and browse for http://localhost:3000. You should see below image on your browser.

If you have any problem see the FAQ section or you may consider filing an issue with your suggestions for improvement.

What's in the box?

  1. React, JSX, ES6, and Flow syntax support.
  2. Preconfigured Redux, Apollo GraphQL client and i18n out-of-the-box. (You can disable Redux/ GraphQL client if you're not using it, it'll leave minimum trace in your build)
  3. Server-side rendering, code-splitting and hot-module replacement configured out-of-the-box.
  4. CSS Modules with autoprefixer, so you won't need to use -webkit or other vendor prefixes.
  5. A build script to bundle JS, CSS, and images for production, with hashes and sourcemaps.
  6. A live development server that warns about common mistakes.
  7. Language extras beyond ES6 like the object spread operator.
  8. Customizable with:
    • Custom Helpers/Sidecars (Server-side)
    • Custom Middlewares (Server-side)
    • Custom React App, Server App, GraphQL client & Redux Implementation, etc.
    • Custom Webpack, Babel & PostCSS Config (Not recommended though, should only be used as escape hatch).
  9. Filesystem as your API (all entry hooks file and folder are under src/_**)*

Authoring Addons

The main advantage of using Treats for developing your React apps is the ability to customize Treats with addons that provides helpers, middlewares, generators, components, custom server apps, etc. You might want to checkout List of available Treats addons to make sure that someone haven't built addons that suits your need. Or you can go to Authoring Addons section, you'll learn more about how to build and share these addons so it can be used by our fellow community member.
