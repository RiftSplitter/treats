FAQs

Is it possible to use Treats without TKP command-line binary?

It is possible to use Treats directly with zero-dependency to TKP command line binary. Please note that with this approach, you'll need to configure your own workspace by yourself (installing NodeJS, Create Treats App packages, etc).

To use Treats as standalone, you'll need NodeJS >= 6.0.0, Please refer to NodeJS website if you haven't installed NodeJS on your local machine. After NodeJS installed on your local machine:

npm install -g create-treats-app
create-treats-app
cd /YOUR/APP/DIRECTORY
yarn start
